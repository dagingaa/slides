##  Operators

````javascript
// +, -, *, /, %
1+1; 1-1; 2*2; 2/2; 4 % 2;

// + does string concatenation
"hello" + " world";
> "hello world"

// + also does type coercion
"hello " + 123;
> "hello 123"

// compound assignments +=, -=, ++, --
var bar = 0; bar++; bar += 2;

// comparison: >, <, <=, >=
1 < 2; 2 > 1; 2 >= 2; 3 <= 4;
````
