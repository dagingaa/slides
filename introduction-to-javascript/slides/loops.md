##  Loops
Similar to other C-family languages

````javascript
while (true) {
  // an infinite loop!
}

// do..while for when you want at least one execution
var input;
do {
  input = get_input();
} while (inputIsNotValid(input))

for (var i = 0; i < 5; i++) {
  // Will execute 5 times
}

for (var bar in bars) {
  // Dangerous!
}
````
