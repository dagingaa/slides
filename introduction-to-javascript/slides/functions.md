##  Functions

Takes 0 or more named parameters. Can be anonymous and named. Can be assigned.
````javascript
var add = function (a, b) {
    return a + b;
};

function subtract(a, b) {
    return a - b;
};
subtract;
> Function...

add();
> NaN

add(1,2,5);
> 3
````
