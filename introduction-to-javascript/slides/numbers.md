##  Numbers

JavaScript does not have integers. Numbers are "double-precision 64-bit format IEEE 754 values".

````javascript
0.1 + 0.2 == 0.30000000000000004
````

It has `Math`.

````javascript
Math.sin(3.5);
var d = Math.PI * r * r;

// parseInt to convert string to integer
parseInt("123", 10)
> 123
parseInt("010", 10)
> 10
````
